== Description ==

The plugin is used to make a block of features for your website. You will be able to easily implement it anywhere using a shortcode. So if you want to make some changes in the block later, they will be applied everywhere on your website where the block is used.

== Installation ==

1. Upload db-tagcloud folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Enjoy

== Frequently Asked Questions ==

= Why is it useful for my website? =

The block of features helps to demonstrate your potential clients the reasons why to buy your products or order your services.
